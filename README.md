# Notes App

Create, Read, Edit, and Delete notes!

# Technologies
* Ruby Version: 2.6.6
* Rails Version: 6.0.3.2

# Included Gems
* [Devise](https://github.com/heartcombo/devise)

# Features
* Create a notebook with a title and body
* Read your existing notebooks
* Edit a notebook
* Delete a notebook
